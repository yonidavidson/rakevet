rakevet

Browse Israel Railways (rail.co.il) from your terminal — search routes, see the next departures, look up stations, platforms and crowding. Zero dependencies.

View on GitHub Install Usage

1.Install

Requires Node.js ≥ 22.6. The CLI is TypeScript run natively — no build step, no npm install.

git clone https://github.com/yonidavidson/rakevet.git
cd rakevet
npm link   # makes the `rakevet` command available everywhere

Prefer not to link? Run it in place with node src/cli.ts ….

2.Usage

rakevet search   <from> <to> [--date YYYY-MM-DD] [--time HH:MM] [-n N] [--lang en|he] [--json]
rakevet next     <from> <to> [-n N] [--lang en|he] [--json]
rakevet stations [query]      [--json]
rakevet refresh

<from> and <to> accept a station id or a name in English or Hebrew.

search

Routes between two stations at a given date and time.

next

Upcoming departures from now (defaults to 5).

stations

List or search the station catalog (name ↔ id, EN + HE).

refresh

Force-refresh the cached station list (cached 30 days).

3.Examples

# Next trains from now
rakevet next "tel aviv savidor" "jerusalem navon"

# A specific date and time, 5 results
rakevet search 3700 680 --date 2026-07-01 --time 08:30 -n 5

# Find a station's id (English or Hebrew)
rakevet stations jerusalem
rakevet stations "באר שבע"

# Machine-readable output for scripting
rakevet next nahariya "beer sheva center" --json

Example output:

Tel Aviv - Savidor Center → Jerusalem - Yitzhak Navon   2026-06-28 (from 13:27)

13:37 → 14:22  (45m, direct)
    13:37 Tel Aviv - Savidor Center (pl. 4) → 14:22 Jerusalem - Yitzhak Navon (pl. 3)   train 745  load •••

load is the API's predicted crowding: · light, •• moderate, ••• busy.

4.Hebrew output

Pass --lang he to print station names, platforms and service messages in Hebrew. The default follows your locale (LC_ALL / LC_MESSAGES / LANG).

Terminals without RTL support. Many terminals (e.g. Ghostty) don't reorder bidirectional text, so Hebrew would print mirrored. rakevet renders Hebrew in visual order so it reads correctly on screen — the trade-off is that copying that Hebrew yields reversed characters. English output is always copy-safe.

5.Notes

This is an unofficial tool, not affiliated with or endorsed by Israel Railways. It relies on the same undocumented public API the website uses and may break if that API changes. Use responsibly.
